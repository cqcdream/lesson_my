# 标签生成器
    它必须以#标签开头。
    所有单词的首字母必须大写。
    如果最终结果超过 140 个字符，则必须返回false
    如果输入或结果为空字符串，则必须返回false
    例子:" Hello there thanks for trying my Kata"  =>  "#HelloThereThanksForTryingMyKata"
        "    Hello     World   "                  =>  "#HelloWorld"
        ""                                        =>  false
